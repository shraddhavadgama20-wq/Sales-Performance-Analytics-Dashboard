# Sales-Performance-Analytics-Dashboard

📌 Overview
This project is an Excel-based data analysis dashboard built using a Kaggle dataset. It analyzes retail and warehouse sales performance across multiple years, focusing on trends, category performance, and top-selling products.
The goal of this project is to transform raw sales data into meaningful business insights using Excel pivot tables and visual dashboards.

🌍 Dataset Overview
The dataset used in this project is sourced from Kaggle and contains detailed monthly sales and inventory movement data across warehouse and retail operations.
It includes:
Item-level sales data
Department/category information
Monthly aggregated sales records

🎯 Objective
The main objectives of this analysis are:
To analyze monthly sales trends over time
To identify top-performing product categories
To determine best-selling products
To understand overall sales distribution patterns

🧰 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning & Transformation
Dashboard Design

📊 Dashboard Features
📈 Monthly Sales Trend Analysis
📦 Category-wise Sales Performance
🏆 Top 10 Products by Sales
📊 KPI Cards for high-level insights

📊 Key Insights
- Sales peaked in July 2020, marking the highest performance period in the dataset.
- Sales show fluctuations across months, ranging from low to high demand cycles.
- Certain months (mid-year periods) consistently show stronger sales performance.
- The Beer category (Corona Extra) is among the top-performing products.
- A small number of products contribute significantly to total revenue.
- The dataset shows potential seasonal trends in sales behavior.
